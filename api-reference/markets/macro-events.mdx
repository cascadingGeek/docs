## Endpoint

https://api.0xmeta.ai/v1/macro_events

### Description

Access macro-level events including regulation changes, institutional listings, and major policy updates — all curated by 0xmeta’s intelligence engine.

### Query Parameters

| Name        | Type    | Description                                              | Default |
| ----------- | ------- | -------------------------------------------------------- | ------- |
| `limit`     | integer | Max number of events returned                            | `10`    |
| `timeframe` | string  | Time window to scan for events (e.g., `1h`, `24h`, `7d`) | `24h`   |
