````mdx
## Endpoint

https://api.0xmeta.ai/v1/markets/categories

### Description

List of all categories that the 0xmeta intelligence graph tracks (e.g., `defi`, `ai_agents`, `nfts`, `layer2`, etc.).

### Response Example

```json

      "category": "defi",
````
