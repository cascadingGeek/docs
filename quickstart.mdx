## Setup your account

<Steps>
  <Step title="Get your API Key">
    Sign up at [dashboard.0xmeta.ai](https://dashboard.0xmeta.ai) to get your API key.
    
    <Note>
      For testing, you can use the demo API key: `test_key_1234567890`
    </Note>
  </Step>

  <Step title="Make your first request">
    Use your API key to verify a payment:
```bash
    curl -X POST https://facilitator.api.0xmeta.ai/v1/verify \
      -H "Content-Type: application/json" \
      -H "X-API-Key: your_api_key" \
      -d '{
        "transaction_hash": "0x...",
        "chain": "base",
        "seller_address": "0x...",
        "expected_amount": "1000000000000000000"
      }'
```

    <Tip>
      Use the `Idempotency-Key` header to prevent duplicate requests
    </Tip>

  </Step>

  <Step title="Handle the response">
    You'll receive a verification response:
```json
    {
      "verification_id": "ver_abc123",
      "status": "verified",
      "transaction_hash": "0x...",
      "chain": "base",
      "verified_amount": "1000000000000000000",
      "verified_at": "2025-01-15T10:30:00Z"
    }
```

    Save the `verification_id` for settlement.

  </Step>

  <Step title="Settle the payment">
    Use the verification_id to initiate settlement:
```bash
    curl -X POST https://facilitator.api.0xmeta.ai/v1/settle \
      -H "Content-Type: application/json" \
      -H "X-API-Key: your_api_key" \
      -d '{
        "verification_id": "ver_abc123",
        "destination_address": "0x..."
      }'
```
  </Step>
</Steps>
