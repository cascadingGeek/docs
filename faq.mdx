## Get Started

No signup required - just start building:

```bash
curl -X POST https://facilitator.api.0xmeta.ai/v1/verify \
  -H "Content-Type: application/json" \
  -d '{
    "transaction_hash": "0x...",
    "chain": "base",
    "seller_address": "0x...",
    "expected_amount": "1000000000000000000"
  }'
```

<Info>
  Start with zero fees during verification. Only pay 0.5% when you successfully
  settle.
</Info>
